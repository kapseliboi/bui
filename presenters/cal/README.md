## Properties
- `date` - accepts a string, dayjss date, or object (ex: {month:3})

## Methods
- `nextMonth()`
- `prevMonth()`
- `goToToday()`

## Events

- `date-changed` - `event.detail` will be the date range

## Slots

- `after-title`
- `before-nav`
- `after-nav`
